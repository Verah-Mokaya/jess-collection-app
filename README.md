<!-- Jess Collection  -->

Jess Collection is a modern eCommerce web application selling hijabs, shoes, and jewellery, designed to serve customers across Africa. The business is proudly based in Hargeisa, Somaliland (Somali Region) and focuses on delivering high-quality fashion items with reliable regional delivery.

<!-- About Jess Collection -->

Jess Collection makes modest fashion and accessories accessible to customers throughout Africa. From elegant hijabs to stylish shoes and beautiful jewellery, the platform offers a smooth, secure, and mobile-friendly shopping experience.

Installation

cd jess-collection

npm install
npm run dev

<!-- Environment Variables -->

VITE_SUPABASE_URL=your_supabase_url
